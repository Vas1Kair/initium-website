↗ Live Market Data (Finnhub)
Not connected
Connect a free Finnhub API key to show real-time stock prices in the ticker band and portfolio table on the live site. Prices auto-refresh every 60 seconds.
  1. Go to finnhub.io/register and create a free account
  2. Copy your API Key from the dashboard
  3. Paste it below and click ↓ Save Settings
Note: the free tier covers US stocks & ETFs. Indices (S&P 500, NASDAQ, etc.) and gold use ETF proxies (SPY, QQQ, GLD…). Leave blank to keep the static saved values.

◇ Payments & Stripe
Connect Stripe to accept subscription payments. Paste your Payment Links from the Stripe Dashboard — no backend needed.
How to set up Stripe
  1. Go to dashboard.stripe.com and create a free account
  2. Go to Payment Links → Create a link for each plan
  3. Copy each link URL and paste it below
  4. Click ↓ Save — the Subscribe buttons on the site will redirect to Stripe Checkout
  5. Stripe handles the payment, sends you the money, and emails the customer a receipt

◎ Notifications & Forms
Get emailed whenever someone submits the contact or access request form on your site. Uses Formspree — free, no backend needed.
How to set up Formspree
  1. Go to formspree.io and create a free account
  2. Click + New Form → give it a name (e.g. "Initium Contact")
  3. Copy the Form Endpoint URL (looks like https://formspree.io/f/xabcdefg)
  4. Paste it below and save — all form submissions will be emailed to you instantly
